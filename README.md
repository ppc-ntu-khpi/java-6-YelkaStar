Умова
Відсортувати квадратну матрицю заданого порядку N у порядку зростання елементів за обходом «зліва-направо, зверху-вниз».
Тобто після сортування найменший елемент має опинитися у лівому верхньому куті [0][0], далі числа зростають уздовж рядка, потім переходять на наступний рядок, і так до правого нижнього кута [N-1][N-1].
Приклад
Вхідна матриця (N = 3):

9  3  7
1  8  2
6  5  4
Результат:
1  2  3
4  5  6
7  8  9

Ідея розв'язання
Завдання реалізовано без явних циклів — лише засобами класу java.util.Arrays та потоків IntStream. Алгоритм складається з трьох кроків:

Розпрямлення матриці у одновимірний масив

java   int[] flat = Arrays.stream(matrix).flatMapToInt(Arrays::stream).toArray();

Сортування одновимірного масиву за зростанням

java   Arrays.sort(flat);

Розкладання відсортованого масиву назад у матрицю порядку N — рядок i це шматок [i·N, i·N + N)

java   Arrays.setAll(result, i -> Arrays.copyOfRange(flat, i * N, i * N + N));
Вихідна матриця при цьому не змінюється — повертається нова.
Структура проєкту

.
├── README.MD
└── src
    ├── MatrixSorter.java       # основний клас: лише логіка (статичний метод sort)
    └── MatrixSorterTest.java   # тестовий клас: main лише перевіряє роботу

MatrixSorter — універсальний, без інтерфейсу користувача. Метод sort публічний; допоміжна перевірка validateSquare — приватна.
MatrixSorterTest — main не містить логіки завдання, лише готує дані, викликає sort і виводить результат.

▶Запуск
bashcd src
javac MatrixSorter.java MatrixSorterTest.java
java MatrixSorterTest
Зауваження щодо типів даних
Елементи зберігаються як int — універсальний вибір для довільного діапазону цілих. Якщо за умовою діапазон значень гарантовано малий (наприклад, 0–255), для економії пам'яті матрицю можна оголосити як byte[][] або short[][], відповідно скоригувавши сигнатуру методу.
